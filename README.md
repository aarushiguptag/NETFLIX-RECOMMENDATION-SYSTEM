# netflix recomendation system
Built a recommendation system using content-based filtering techniques. Processed and analyzed datasets to generate personalized recommendations. Tech Stack: Python, Pandas, NumPy



Overview
This project is a content-based movie recommendation system that suggests similar movies based on user-selected titles. It uses movie metadata and similarity measures to generate personalized recommendations.

 Features
- Content-based recommendation
- Movie similarity prediction
- Fast recommendation generation
- Simple and interactive notebook implementation

Tech Stack
- Python
- Pandas
- NumPy
- Scikit-learn (if used)

  Dataset
Dataset used:
- TMDB 5000 Movies Dataset
(Source: Kaggle)


 Project Workflow
1. Data Collection
2. Data Cleaning
3. Feature Engineering
4. Vectorization
5. Similarity Calculation
6. Recommendation Generation
